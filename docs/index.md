# eBPF Docs

Welcome to the eBPF Docs! eBPF is an amazing technology which enables its users to extend the functionality of operating systems in a fast and secure way. eBPF is powerful, but also very complex, especially for newcomers. 

This is a collection of technical documents regarding eBPF. If you are new to eBPF, it is recommended to orient yourself using the [What is eBPF](https://ebpf.io/what-is-ebpf) article and come back to explore different aspects in depth.
